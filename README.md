## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn
- Git

### Installation

```bash
# Clone the repository
git clone https://github.com/fincloud9/PMWAVECORE.git

# Navigate to the project directory
cd PMWAVECORE

# Install dependencies
npm install

# Start development server
npm run dev
```
